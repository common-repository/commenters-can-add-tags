Commenters can add tags allows commenters to add tags to a post just by adding a prefixed word in a comment. Prefix is # by default.


== Description ==



This plugin allow commenters to add a tag to a post, just by using a customized prefix (by default #) before words in comments.


I18N:

This plugin is I18N ok.


== Installation ==

1. Unload plugin, unzip it to /wp-content/plugins/ directory
2. Go to plugins Tab and activate it.
3. Customize it in email-mentioned tab.
